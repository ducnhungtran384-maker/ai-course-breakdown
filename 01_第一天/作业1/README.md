# 10_职业规划咨询系统_20260406

## 项目简介

AI驱动的职业规划咨询系统，为中山大学临床医学大一学生提供转专业到软件工程的完整规划方案。

## 核心内容

- **咨询对话实录**：完整记录AI顾问与学生的多轮对话，从焦虑分析到路径规划
- **规划报告**：包含10个章节的详细规划，涵盖学生画像、筹码评估、规划方案、风险评估等
- **可视化页面**：3000+行单页面应用，精美UI呈现报告和对话，支持分享和下载

## 方法论框架

基于28个世界顶级理论框架：
- Bourdieu资本论
- Granovetter弱关系理论
- 7阶段对话流程（暖场→画像→资本盘点→节点地图→路径分裂→减枝决策→行动输出）
- 杠铃策略：90%精力主路径 + 10%精力备选路径

## 技术实现

- 纯HTML/CSS/JavaScript（无框架依赖）
- Markdown解析和渲染
- 响应式设计，支持移动端
- 单文件部署，可离线查看
