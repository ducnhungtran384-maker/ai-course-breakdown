## 🚀 快速开始

### 本地预览

1. 克隆仓库
```bash
git clone https://github.com/ducnhungtran384-maker/ai-course-breakdown.git
cd ai-course-breakdown
```

2. 用浏览器打开 `index.html` 即可

### 部署到 GitHub Pages

1. Fork 或克隆本仓库
2. 进入仓库的 **Settings** → **Pages**
3. Source 选择 **Deploy from a branch**
4. Branch 选择 **main**，目录选择 **/ (root)**
5. 点击 **Save**
6. 等待几分钟后访问 `https://ducnhungtran384-maker.github.io/ai-course-breakdown/`

## 🎯 使用说明

1. **浏览内容**：滚动页面查看道法术器四层拆解
2. **播放音乐**：点击右下角的黑胶唱片开始播放背景音乐
3. **查看动画**：滚动时内容会平滑淡入，粒子在背景中动态连接

## 🛠️ 技术栈

- **纯原生技术**：HTML5 + CSS3 + JavaScript
- **Canvas API**：粒子系统动画
- **Web Audio API**：音频播放控制
- **Intersection Observer API**：滚动动画
- **CSS Animations**：旋转、悬浮、发光效果
